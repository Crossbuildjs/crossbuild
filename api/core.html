CustomCheckFunction

CustomCheckFunction: (interaction: ReceivedInteraction, component: Component) => Promise<GeneratedMessage | null>

Type declaration

    • (interaction: ReceivedInteraction, component: Component): Promise<GeneratedMessage | null>

    • This is a function that is called to check if a Component should be run or not. If the function returns a GeneratedMessage, the message will be sent to the channel that the interaction was received in. Otherwise, the function returns null and the component will be run.

      Parameters

      • interaction: ReceivedInteraction
      • component: Component

      Returns Promise<GeneratedMessage | null>
