This guide will teach you how to:

- Install CrossBuild
- Setup your bot to connect to both Discord and Guilded
- Create commands
- Add your own events

## Before you begin

Before you begin, you'll need to have a basic grasp of Javascript and Node.js. If you don't, we recommend you check out [JavaScript.info](https://javascript.info) to learn the basics of Javascript, before attempting to use this package.
